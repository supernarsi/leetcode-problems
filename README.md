# leetcode-problemset

本仓库包含了 LeetCode 上的题目解答，使用 Golang 语言编写。每道题目都包含了一个 problem.go 文件，其中包含解答代码。同时，每道题目都包含了一个 problem_test.go 文件，用于测试解答。

## 使用

### 1. 克隆仓库

```bash
git clone https://github.com/supernarsi/leetcode-problems.git
```

### 2. 进入目录

```bash
cd problems
```

### 3. 运行测试

```bash
go test -v
```
